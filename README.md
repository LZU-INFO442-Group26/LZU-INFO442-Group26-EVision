# EVision

Cross-Platform Analysis of Consumer Discussions on Chinese EV Brands

EVision is a data science project focused on analyzing consumer discussions about major Chinese EV brands across multiple Chinese social media platforms, including Bilibili, Douyin, and Xiaohongshu. 

We aims to explore how different online communities discuss EV brands such as Tesla, BYD, and Xiaomi Auto through sentiment analysis, topic analysis, and cross-platform comparison. By combining social media data collection, NLP techniques, visualization, and dashboard deployment, the project seeks to identify platform-specific consumer concerns and discussion patterns.
